Readme

This folder contains the scripts to run the cell-cell communication analysis using cellphoneDB v4

cellphoneDB1* files have the code to create the two files needed for cellphoneDB v4, method degs_analysis. These notebooks run the run_cellphone* scripts.

run_cellphone* files have the code to run the actual analysis using cellphone.
ktplots_exploration is the code to explore the cellphoneDB output using KTplots.
