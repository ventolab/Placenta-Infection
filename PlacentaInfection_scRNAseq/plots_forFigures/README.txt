README

This folder has the scripts to generate the single-cell plots included in the article.

1) cellphoneDB1* and cellphoneDB2* are scripts to create the dotplots used to represent the ligand-receptor interactions predicted by cellPhoneDB v4.

	Scripts to plot the matching Heatmaps is in:
		../diffGeneExpression/fibroEndo/src/heatmaps*
		../diffGeneExpression/macrophages/src/heatmaps*
		../diffGeneExpression/trophoblast/src/heatmaps*

2) plotsForSubmission_* notebooks have the code to create all other plots such as UMAPS, violin plots, receptor heatmaps, etc.
